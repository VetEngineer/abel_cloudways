# FileZilla로 워드프레스에 파일 업로드하기

## 📥 1단계: FileZilla 다운로드 및 설치

1. [FileZilla 공식 사이트](https://filezilla-project.org/) 접속
2. **FileZilla Client** 다운로드 (무료)
3. 설치 진행

## 🔑 2단계: FTP 접속 정보 확인

호스팅 업체(카페24, 가비아, Bluehost 등)에서 제공한 FTP 정보 확인:

```
FTP 호스트: ftp.yourdomain.com (또는 IP 주소)
사용자명: your_username
비밀번호: your_password
포트: 21 (일반 FTP) 또는 22 (SFTP)
```

> **💡 Tip**: 호스팅 관리 페이지 또는 가입 시 받은 이메일에서 확인 가능

## 🌐 3단계: FileZilla로 서버 접속

### 방법 A: 빠른 연결
1. FileZilla 실행
2. 상단 입력란에 정보 입력:
   - **호스트**: `ftp.yourdomain.com`
   - **사용자명**: FTP 계정명
   - **비밀번호**: FTP 비밀번호
   - **포트**: `21` (또는 `22`)
3. **빠른 연결** 클릭

### 방법 B: 사이트 관리자 (추천)
1. **파일** → **사이트 관리자** (Ctrl + S)
2. **새 사이트** 클릭
3. 사이트 이름: `내 워드프레스`
4. 정보 입력:
   - **프로토콜**: FTP 또는 SFTP
   - **호스트**: `ftp.yourdomain.com`
   - **포트**: `21` (FTP) 또는 `22` (SFTP)
   - **로그온 유형**: 일반
   - **사용자**: FTP 계정명
   - **비밀번호**: FTP 비밀번호
5. **연결** 클릭

## 📂 4단계: 워드프레스 디렉토리로 이동

연결 성공 후:

1. **오른쪽 창** (원격 사이트)에서 워드프레스 루트 디렉토리 찾기:
   - `/public_html/` (가장 일반적)
   - `/www/`
   - `/htdocs/`
   - `/home/username/public_html/`

2. 해당 폴더 더블클릭하여 이동

## 📁 5단계: geo-landing 폴더 생성

1. 오른쪽 창(서버)에서 빈 공간 **우클릭**
2. **디렉토리 만들기** 선택
3. 폴더명 입력: `geo-landing`
4. **확인** 클릭
5. 생성된 `geo-landing` 폴더 더블클릭하여 진입

## ⬆️ 6단계: 파일 업로드

### 왼쪽 창에서 로컬 파일 찾기
1. **왼쪽 창** (로컬 사이트)에서 다음 경로로 이동:
   ```
   e:\실전코딩\안티그래비티\웹사이트제작\aeoglobal-landing
   ```

### 파일 업로드
1. 다음 파일들을 **선택** (Ctrl + 클릭으로 다중 선택):
   ```
   ✅ index.html
   ✅ styles.css
   ✅ script.js
   ```

2. 선택한 파일을 **오른쪽 창으로 드래그** 또는 **우클릭** → **업로드**

3. `assets` 폴더도 **드래그하여 업로드**
   - 폴더 안의 모든 파일이 자동으로 업로드됩니다

### 업로드 진행 확인
- 하단 창에서 업로드 진행 상황 확인
- "전송 완료" 메시지 확인

## ✅ 7단계: 업로드 확인

오른쪽 창(서버)에 다음 구조가 보여야 합니다:

```
/public_html/geo-landing/
├── index.html
├── styles.css
├── script.js
└── assets/
    ├── hero-iphone-mockup.png
    ├── comparison-visual.png
    └── solution-icons/
        ├── icon-geo.png
        ├── icon-gmb.png
        ├── icon-multilang.png
        └── icon-aibot.png
```

## 🔐 8단계: 파일 권한 설정 (선택사항)

권한 오류가 발생하는 경우:

1. 업로드한 파일 **우클릭**
2. **파일 권한** 선택
3. 숫자 값 입력:
   - **파일**: `644`
   - **폴더**: `755`
4. **확인** 클릭

## 🌍 9단계: 브라우저에서 확인

브라우저 주소창에 입력:
```
https://yourdomain.com/geo-landing/
```

페이지가 정상적으로 표시되면 성공! 🎉

## 🔧 문제 해결

### "연결 실패" 오류
- FTP 정보가 정확한지 확인
- 포트 번호 확인 (21 또는 22)
- 방화벽 설정 확인

### "권한 거부" 오류
- FTP 계정에 쓰기 권한이 있는지 확인
- 호스팅 업체에 문의

### 이미지가 안 보여요
- `assets` 폴더가 제대로 업로드되었는지 확인
- 파일명 대소문자 확인 (Linux 서버는 대소문자 구분)

### 404 에러
- `index.html` 파일이 `geo-landing` 폴더 안에 있는지 확인
- URL이 정확한지 확인

## ⚡ 빠른 팁

- **Ctrl + 드래그**: 여러 파일 한 번에 업로드
- **F5**: 폴더 새로고침
- **북마크**: 자주 사용하는 경로 저장
- **대기열**: 대용량 파일은 대기열에 추가 후 일괄 업로드
